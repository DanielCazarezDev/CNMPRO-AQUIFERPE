# Aquifer PE is a freelance platform specialized in the civil engineering and construction industry. Aquifer PE facilitates the hiring of specialized professional engineers for infrastructure projects and provides a solution for firms to extend the workforce through access to contract hiring of specialized engineers per project needs.

Aquifer PE is an MVP and due to the proprietary nature of the code I can only provide code snippets of my code. 

This Repository consists of my contribution to the creation of CNM PRO / Aquifer PE. I was responsible for the creation of the File Manager with administrator access and the front end for the User Dashboard features.
